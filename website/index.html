Installation steps

Keep the rollout predictable.

Windows

  1. Download the official `.exe` installer from the Windows release card.
  2. Run the installer and approve any standard Windows security prompts.
  3. Launch Nemorax from the desktop shortcut or Start menu.
  4. Use guest mode for quick access or sign in if persistent history is enabled for your rollout.

Android

  1. Download the official `.apk` from the Android release card.
  2. Allow installation from the approved source if your device asks for permission.
  3. Open the installed app and verify the welcome screen, safe top area, and drawer navigation.
  4. Confirm session behavior if you are testing guest mode.

iPhone and web

  1. Open the official Nemorax web deployment URL in Safari or another supported browser.
  2. Confirm that the mobile layout, safe top spacing, and drawer behavior render correctly.
  3. Use the site directly in-browser; no installed iOS build is currently part of this release path.
  4. Report layout or interaction issues through the feedback workflow.

Support and contact

Launch feedback is part of the product workflow.

Use the in-app feedback feature for product issues, UI problems, and rollout observations. When reporting a problem, include the device, platform, browser or app build, and the exact steps that caused it.

Recommended report details

Device model, operating system, browser or installed build, guest or signed-in mode, reproduction steps, expected result, and actual result.

Operational guidance

  • Use the in-app Feedback action for the fastest product reporting path.
  • Use the Windows installer for managed desktop rollout.
  • Use the Android APK for direct device testing.
  • Use browser deployment for iPhone access and web QA.
